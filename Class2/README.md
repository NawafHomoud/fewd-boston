
# Class 2 Materials

Class 2 covers a lot!  We're going to be learning about some of the fundamental ways that elements in HTML behave and how that affects how we style them.  Additionally, we're going to cover images, margins, padding and units of measurement.  There's no shortage of stuff to learn today, so buckle up and get ready for a fun ride!

## Objectives

Before moving into next week, make sure you're comfortable with the following:

<ul>
  <li>Add images to the page and background</li>
  <li>Add padding and margins to elements</li>
  <li>Use fixed and relative units of measurement</li>
  <li>Identify block and inline elements</li>
  <li>Understand the CSS box model</li>
  <li>Add colors with transparency</li>
</ul>
